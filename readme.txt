Ejecución de la Aplicación:
- Abrir el archivo de la aplicación con el editor de texto de su preferencia.
- Entrar a la carpeta PUICAGUIFuentes.
- Ejecutar el archivo main.py.
- Presionar botón "Subir Archivo" y seleccionar una prueba en formato txt.
- Presionar boton "Resolver" para visualizar las respuestas.
- Se puede manipular este resultado en el archivo "soluciones.txt" dentro de la carpeta "soluciones"

Requisitos:
- Tener instalado Python en la máquina
- Tener instaciado la aplicación de MiniZinc en el PATH del sistema operativo
- Instalar libreria tkinter
